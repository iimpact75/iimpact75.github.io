Font Choices

The font for all the headers will be the bold 'ACME'. Gudea font was chosen for the prargraph text. See below for the font CSS code listings and API link.

  • font-family: 'Acme', sans-serif;
  • font-family: 'Gudea', sans-serif;
  • link href="https://fonts.googleapis.com/css?family=Acme%7CGudea&display=swap" rel="stylesheet"
