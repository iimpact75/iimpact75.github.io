Color Usage

Greens and Yellows were chosen since they are colors that work well together. The green suggests nature and the yellow happiness which is why this is perfect for a weather oreinted website.

Color Usage
Dark Green This is meant as an accent color for boarders like for sections, images, and font colors.
Medium Green Medium green will be used for the headers and footer as a way to frame the content properly.
Light Green Light green is used for the Primary Navigation to grab the attention. This can be used as a font color in conjunction with the dark green for a nice sharp contrast.
Gold This is a great font color and boarder trim. It works well with the green colors
Tan This is meant to be an alternative background color to white. This may help in differentiating one section from another.
